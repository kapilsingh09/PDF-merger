# 📄✨ PDF Merger 

> 🚀 **A modern, intuitive web application that seamlessly merges multiple PDF files into one unified document**

Transform your document workflow with our elegant PDF merger that combines powerful functionality with a beautiful, responsive design. Built with modern technologies and user experience at its core.

---

## 🌟 Why PDF Merger ?

In today's digital world, managing multiple PDF documents can be overwhelming. Whether you're a student compiling research papers, a professional preparing reports, or anyone dealing with scattered PDF files, our app provides the perfect solution with:

- **🎯 One-Click Simplicity** - Upload, arrange, and merge with minimal effort
- **🎨 Beautiful Interface** - Clean, modern design that adapts to your preferences
- **⚡ Lightning Fast** - Instant processing with real-time feedback
- **🔒 Privacy First** - All processing happens securely without storing your files

---
## ✨ Key Features

### 📤 **Smart File Management**
- 🔄 **Multi-PDF Upload** with intelligent duplicate detection
- 📋 **File Overview** showing name, size, and upload order
- 🗑️ **Flexible Removal** - delete individual files or clear all at once
- 📊 **Real-time Stats** displaying total file count and combined size

### 🎨 **Enhanced User Experience**
- 🌙 **Dark/Light Mode Toggle** with persistent theme memory
- 🎬 **Smooth Animations** powered by Framer Motion
- 📱 **Fully Responsive** design that works on all devices
- ♿ **Accessibility First** with proper ARIA labels and keyboard navigation

### ⚡ **Powerful Processing**
- 🛠️ **Server-side PDF Merging** using industry-standard `pdf-lib`
- 📥 **Instant Download** of your merged document
- 🔔 **Smart Notifications** with success/error feedback
- 🛡️ **Robust Error Handling** for a smooth experience

## 🚀 Quick Start Guide

### 🔧 **Installation Steps**

#### 1️⃣ **Clone the Repository**
```bash
git clone https://github.com/your-username/pdf-merger-pro.git
cd pdf-merger-pro
```

#### 2️⃣ **Setup Backend Server**
```bash
cd backend
npm install
npm start
```
🌐 Backend server will be running at `http://localhost:5000`

#### 3️⃣ **Setup Frontend Application**
```bash
cd frontend
npm install
npm run dev
```
🎨 Frontend will be available at `http://localhost:5173`

#### 4️⃣ **Start Merging! 🎉**
Open your browser and navigate to the frontend URL to start using the application.

## 💡 How It Works

### 🔄 **The Magic Behind the Scenes**

1. **📤 Upload Process**
   - Files are validated for PDF format
   - Duplicate detection prevents confusion
   - Real-time file information display

2. **🔧 Server Processing**
   - Secure multipart form handling
   - In-memory PDF manipulation
   - Efficient binary data streaming

3. **📥 Download Delivery**
   - Instant merged PDF generation
   - Clean filename with timestamp
   - Automatic browser download trigger
